# Code guidelines
Whenever designing views or pages, make sure to use responsive design principles to ensure compatibility across different devices and screen sizes.
There exist sever utility classes in lib/core/utils to help with responsiveness, along with more stuff.

# UI Design
Always adhere to material 3 design principles when creating UI components.

# Security

When working with sql tables, always enable RLS (Row Level Security) to ensure data security and proper access control. Make sure only authorized users can access or modify sensitive data.
